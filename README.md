# Call Log
Suppose you are an admin of a Multinational company. You need a report for
customer care phone call log based on the status ​ In-call, hold, call back, do not
call​ .

## Requirements:
1. Use ​ Laravel 7+
2. Use VueJS​ - (Please don't use jquery)

## Features:
1. List of 500 phone call logs in a grid where fields are call date, phone number, call duration & status.
2. The same page gives a timeline chart
        a. Date vs count Number of calls
        b. Date Range filter
3. Filter based on status & date range on the same page.
4. Grid & Timeline chart will be affected based on the dropdown status and date range selected.
5. For data, Axios call using a database ​.
